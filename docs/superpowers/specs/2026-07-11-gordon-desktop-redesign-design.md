# Дизайн «Пульт Гордона» v0.5 — Неон Cyber

Редизайн десктопного дашборда `gordon_desktop.py` (PySide6) под максимально
сочный, современный неон-кибер стиль. Все текущие функции сохраняются
работоспособными; меняется только визуал и раскладка окна.

## Контекст

Текущее состояние (`gordon_desktop.py`, v0.4): нативное PySide6 окно, плоская
тёмно-серая тема (`#0f1115` фон, `#4f9dff` синий акцент), без градиентов,
свечения и глубины. Работает, но выглядит «сухо». Назар хочет сочно и по-новому.

Решения, согласованные в брейншторме:
- **Направление:** Неон Cyber (глубокий чёрный фон, неоновый градиент
  фиолет→циан→роза, свечение на элементах).
- **Объём:** Полная переделка лейаута (header + sidebar + центр) при сохранении
  ВСЕХ функций дашборда.
- **Динамика:** Пульс индикатора статуса + hover-свечение кнопок.
- **Реализация:** Гибрид QSS + кастомные виджеты (QPainter для glow-градиентов,
  QGraphicsDropShadowEffect для свечения, QPropertyAnimation для пульса).

Ограничения (из глобального CLAUDE.md): стек не меняем (нативный PySide6,
без веба), БД `outreach.db` не трогаем, логику запуска агентов не ломаем.

## Палитра

- Фон окна/панелей: `#070710` (чистый чёрный), лёгкий вертикальный градиент
  к `#0d0a1a` сверху вниз.
- Неон-бренд-градиент: `#a855f7` (фиолет) → `#22d3ee` (циан) → `#f472b6` (роза).
- Текст: `#e6e6e6` (основной), `#8a93a3` (приглушённый).
- Статусы (неоновые, узнаваемые):
  - `pending` `#64748b` (серо-голубой)
  - `sent` `#4f9dff` (ярко-циан/синий)
  - `replied` `#22c55e` (неон-зелёный)
  - `hired` `#f5c518` (золотой неон)
  - `rejected` `#ff4d6d` (неон-красный)
  - `bounced` `#fb923c` (неон-оранжевый)
- Панели второго уровня (полупрозрачные): `rgba(255,255,255,0.04)` поверх
  градиента фона с тонким неон-бордером.
- Шрифты: `Segoe UI` (текст/цифры), `Consolas` (лог/код) — оставляем как сейчас.

- **Header:** слева неон-иконка щита `🛡` в glow-рамке + градиентный заголовок
  «ПУЛЬТ ГОРДОНА» + мелкая версия `v0.5`; справа `StatusIndicator` пульсирующий.
- **Sidebar (фикс ~240px, вертикальный):** неон-кнопки агентов:
  `▶ Запустить Гордона`, `⏹ Стоп`, `⚠ Форсированная отправка (вне окна!)`,
  `🕸 Парсинг сайтов`, `📥 Просмотр ответов`. Снизу — блок инфо про гард
  (как сейчас, в приглушённом стиле).
- **Центр:** сверху ряд `NeonCard` (Total/Pending/Sent/Replied/Hired/Rejected/
  Bounced/Конверсия/Заработано). Ниже `QSplitter` (вертикаль): фильтры + таблица
  (вверху), живая лента лога (внизу). Фильтры — неон-поля (`QComboBox`/`QLineEdit`).

## Кастомные виджеты (гибрид QSS + QPainter)

- **`NeonCard`** (наследник `QFrame`/`QWidget`): `paintEvent` рисует
  `QPainter`'ом скруглённый прямоугольник с полупрозрачным неон-градиентом фона
  + glow-бордер (радиальное свечение по контуру через `QGraphicsDropShadowEffect`).
  Число в неоне, вспышка подсветки при изменении значения (`set_value`).
- **`NeonButton`** (наследник `QPushButton`): база — QSS с неон-градиентом;
  `enterEvent` усиливает `QGraphicsDropShadowEffect` (blur + opacity),
  `leaveEvent` возвращает; `mousePress` — короткая вспышка. Для кнопок-действий
  (stop/reject/bounce) — цвет свечения = цвет статуса.
- **`StatusIndicator`** (header): `QPropertyAnimation` пульсирует `opacity`/`blur`
  glow-эффекта. Состояние «работает» — циан-пульс; «не запущен» — спокойный серый.
- **Таблица/лог:** обычный QSS (логика `read_sites`, `refresh_table`, `tail_log`
  не трогается), но с неон-бордерами и подсветкой ячейки статуса цветом из палитры.
- **Свежеотправленные (`_fresh_sent`):** строки подсвечиваются неон-зелёным и
  плавно затухают (анимация фона через `QPropertyAnimation`).

## Что НЕ трогаем (функционально)

- Логика БД: `get_conn`, `read_stats`, `read_sites`, `read_site` — без изменений.
- Запуск Гордона: `GordonRunner` (QThread + subprocess), `start_gordon`,
  `start_forced`, `stop_gordon`, `on_runner_finished` — те же слоты, новые стили.
- `track.py` вызовы (`run_track`), окна `SiteDetailDialog` и `open_replies`
  (парсинг/IMAP) — рабочие, обновляется только обёртка под новый стиль.
- Таймеры: стата 3с, таблица 3с, poll_sent 2с, лог 1с — без изменений.
- Стек: PySide6, запуск `python gordon_desktop.py`. Без веба.

## Структура файла

Один файл `gordon_desktop.py` (как сейчас, единый модуль). Новые классы
(`NeonCard`, `NeonButton`, `StatusIndicator`) добавляются рядом с `StatsCard`.
Палитра выносится в константы вверху. Функции БД и runner без изменений.

## Проверка (verification)

1. `python gordon_desktop.py` открывается без ошибок (проверить, что PySide6
   установлен; если нет — `pip install PySide6`).
2. Неон-стиль виден: чёрный градиентный фон, неон-карточки, неон-кнопки sidebar.
3. `StatusIndicator` пульсирует при запуске Гордона (циан), спокоен когда стоит.
4. Hover по кнопкам усиливает свечение; click — вспышка.
5. Карточки показывают реальные цифры из `outreach.db` (215 сайтов, 103 sent и т.д.).
6. Таблица/фильтры/лог/окна деталей и ответов работают как раньше.
7. Запуск Гордона/парсинга/проверки почты из новых кнопок sidebar не сломан.
8. Свежеотправленные строки подсвечиваются неон-зелёным и затухают.
